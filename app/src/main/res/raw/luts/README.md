# Luma Camera 内置 LUT

将 LUT 文件放入此文件夹，编译后自动成为内置滤镜。

## 支持格式
- `.cube` - Adobe 标准格式
- `.3dl` - Autodesk 格式

## 命名规范
文件名即为滤镜显示名称。

示例：
- `日系清新.cube` → 显示为「日系清新」
- `电影感.cube` → 显示为「电影感」
- `黑金风格.cube` → 显示为「黑金风格」

## 推荐尺寸
- 17×17×17 - 文件小，性能最优
- 33×33×33 - 推荐，质量和性能平衡
- 65×65×65 - 最高质量，文件较大

## 添加步骤
1. 将 `.cube` 或 `.3dl` 文件复制到此目录
2. 重新编译项目
3. 内置滤镜自动生效
